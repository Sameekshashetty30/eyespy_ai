# EyeSpy AI – Eye Disease Detection System

## Overview

EyeSpy AI is a deep learning–based application that detects eye diseases from retinal images.
The system uses a trained deep learning model to classify eye conditions and assist in early diagnosis.

## Features

* AI-based eye disease detection
* Image upload for prediction
* Deep learning model for classification
* User-friendly interface

## Tech Stack

* Python
* TensorFlow / PyTorch
* Flask / FastAPI
* HTML, CSS, JavaScript
* Machine Learning / Deep Learning

```

Navigate to the project folder:

```
cd eyespy_ai
```

```
